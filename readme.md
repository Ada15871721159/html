### HTML基础学习笔记

**HTML**:超文本标记语言，HTML不是一种编程语言，而是一种标记语言

**常用标签**

| 标题标签             | <h1>...</h1>  <h2>...</h2>  ...  <h6>...</h6> |
| -------------------- | --------------------------------------------- |
| 段落标签             | <p>...</p>                                    |
| 水平线标签           | <hr/>                                         |
| 换行标签             | <br/>                                         |
| div标签（用于布局）  | <div>...</div>  (一行只能放一个）             |
| span标签（用于布局） | <span>...</span>  (一行能放多个）             |
| 文字加粗             | <b>...</b>  <strong>...</strong>              |
| 文字斜体             | <i>...</i>  <em>...</em>                      |
| 文字加删除线         | <s>...</s>  <del>...</del>                    |
| 文字加下划线         | <u>...</u>  <ins>...</ins>                    |
| base标签             |                                               |
